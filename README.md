前期准备：
    
    修改api.conf：
    corpcode = 公司标识
    host = 公司地址

启动说明：

    无python环境，双击study.exe
    
    python环境下，双击study.bat
    
版本亮点：

    1、python3环境
    2、无需python环境即可运行
    3、GUI可视化界面操作

未来新功能：
